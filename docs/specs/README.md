# canon Roadmap Specs

Implementation specs for the next five canon releases. Each spec is written as a codeplan-style slash-command brief: user surface, scope, file changes, validation, and launch notes.

## Sequence

1. `01-look-back.md` — `/canon:look-back`, mines recent work for reusable skills, subagents, and automations.
2. `02-protected-sections.md` — protected Markdown regions plus diff checks for canon and agent-verify.
3. `03-canon-optimize-alpha.md` — `/canon:optimize`, measured bounded edits for skill files.
4. `04-codex-port.md` — canon for Codex, with portable templates and runtime adapters.
5. `05-autobrowse-skill-graduation.md` — Autobrowse-style strategy iteration and skill graduation.
